# OpenClaw Skills 文档格式规范

## 文件命名
- 格式: `skills/YYYY-MM-DD.md`
- 示例: `skills/2026-03-09.md`

## 文档结构

```markdown
# OpenClaw Skills 每日动态 - YYYY-MM-DD

> 每天自动抓取整理，记录 OpenClaw Skills 最新动态

---

## 今日热门 Skills 🔥

### 1. Skill 名称
- **来源**: 来源平台/URL
- **发布时间**: YYYY-MM-DD
- **热度**: 🔥🔥🔥🔥🔥 (1-5星)
- **功能简介**: 50-100 字功能描述
- **是否已安装**: ✅/❌
- **相似技能**: 名称（差异说明）
- **安全评估**: ✅安全/⚠️需审查/❌风险

---

## 今日新增 Skills

| # | 名称 | 来源 | 功能 | 已安装 |
|---|------|------|------|--------|
| 1 | ... | ... | ... | ✅/❌ |

---

## Skills 详细列表

### 1. Skill 名称
- **来源**: 来源
- **发布时间**: 日期
- **链接**: [查看原文](URL)
- **功能**: 详细功能描述
- **安装命令**: `clawdhub install xxx` 或自定义
- **已安装**: 是/否
- **相似技能**: 
  - xxx: 差异说明
- **安全评估**: 
  - 代码来源: 自创/社区贡献/未知
  - 风险检查: ✅ 无明显风险 / ⚠️ 建议人工审查 / ❌ 存在风险
- **💡 简评**: 50字左右的个人看法

---

## 数据来源

- ClawdHub: https://clawdhub.com
- OpenClaw 论坛: https://discord.com/invite/clawd
- GitHub 搜索: openclaw skills
- 其他: ...

---

*更新时间: YYYY-MM-DD HH:MM:SS*
```

## 每条 Skill 描述要求

| 字段 | 要求 |
|------|------|
| 名称 | 英文名/中文名 |
| 来源 | 平台 + URL |
| 功能简介 | 50-100 字 |
| 是否已安装 | 检查 ~/openclaw-cn/skills/ |
| 相似技能 | 对比已安装技能 |
| 安全评估 | 代码来源 + 风险检查 |
| 简评 | 50 字个人看法 |
